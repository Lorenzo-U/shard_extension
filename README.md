# SHARD extension


Forget the classic and static "New Tab" screen of your browser. SHARD isn't a simple extension but a web dashboard, an operating system integrated directly into Chrome.

![1 1](https://github.com/user-attachments/assets/41fedddb-bd42-4045-afab-1961656a831a)


## Features

Instead of opening dozens of tabs to check the latest news, stocks, or the weather, SHARD allows you to have everything under control directly from a single page. Here are some of the main features:

* **The "Shards"**: I like to call them fragments of the web. They are designed to extract text in real-time from any site by holding down Ctrl and left-clicking on the element. The background system constantly queries the site bypassing the cache Pragma: no-cache) to guarantee you up-to-the-second data.
* **The "Portals"**: You can also insert entire websites and navigate them within floating and resizable windows. Leveraging declarativeNetRequest rules, SHARD surgically removes headers like x-frame-options, allowing you to bypass browser embedding blocks.
* **Native Widgets**: Includes local mini-applications written in JS. Among these, you will find the universal search bar (Google, Bing, DuckDuckGo, Ecosia), the clock (with multiple time zones), the weather, the battery, notes, to-do lists, and Spotify.
* **Edit Mode**: By clicking on the pencil icon, you can customize your layout. Drag the title bar to move the windows, resize them from the bottom right corner, or delete them.

## Installation

Currently, the extension is not published on the Chrome Web Store. It must be installed manually in Developer Mode (Unpacked):

1. **Download the files**: Download the release ZIP file and extract it into a safe folder on your computer. 
2. **Manage Extensions**: Open your Chromium-based browser (Besides Chrome, you can also use Edge, Brave, or any other compatible browser), type chrome://extensions (or edge://extensions and so on depending on the browser) in the address bar, and press Enter.
3. **Enable Developer Mode**: Find the developer mode toggle, generally located in the top right corner, and turn it on.
4. **Load SHARD**: Click on the load unpacked button and simply select the folder you extracted in the first step
5. **Remove the footer**: If you have a footer right-click on it and select "don't show the footer". From now you'll get the full-screen interface.
6. **Done**: Open a new tab to explore your new dashboard!
